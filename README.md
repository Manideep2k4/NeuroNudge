NeuroNudge is an AI-powered habit tracking and productivity web application designed to help users build better habits through behavioral nudges, mood tracking, and focus sessions.

##  Features
- Habit creation, tracking, and archiving
- Daily streak monitoring
- Mood logging with ML-based sentiment analysis
- Pomodoro focus timer with habit tagging
- Habit & productivity calendar visualization
- Personalized feedback based on user behavior

##  Tech Stack
- Backend: Django, Flask (REST APIs)
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Machine Learning: Scikit-learn, NLTK
- Database: SQLite, MySQL
- Tools: Git, GitHub, VS Code

##  Use Case
Designed for students and professionals to improve consistency, mental awareness, and productivity using data-driven insights.

##  Future Enhancements
- Personalized AI nudges
- Mood trend analytics
- Recommendation engine
